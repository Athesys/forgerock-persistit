AdminUI Panels Overview

AdminUI displays a tabbed view with the following panels:

Summary Panel View the Persistit version and license information, buffer pool utilization, prewrite journal buffer status and utilization, sizes and growth parameters of each volume, and other management information.
Buffers Panel View the detailed state of the Persistit buffer pool. Select specific buffer sizes and states and sort the information by any column of the table.
Trees Panel View information about trees examine the data contained in a Tree.
Tasks Panel Start and monitor potentially long-running utility tasks, including tasks to export or import data to or from a save file, or to check the integrity of the B-tree structure of one or more trees.

All tables presented within these panes can be sorted on any column. Click on a column header to toggle between ascending, descending and unsorted order based on the values of that column.
